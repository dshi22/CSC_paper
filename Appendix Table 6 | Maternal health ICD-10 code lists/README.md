Unplanned maternal hospital admissions with any of the specified International Statistical Classification of Diseases and Related Health Problems, 10th Revision (ICD-10) codes were classified as adversity-related or mental health-related. Mothers with hospital admissions for chronic conditions were classified as having a chronic condition.
This appendix comprises three CSV files containing ICD-10 code lists adapted from Herbert et al. and Hardelid et al. The “Adversity-related injuries” file is based on code lists from Herbert et al. The “Mental health problems” and “Chronic health conditions” files are based on code lists from Hardelid et al., with codes overlapping with the “Adversity-related injuries” file excluded.
